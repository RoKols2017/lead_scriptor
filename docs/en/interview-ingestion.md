# Interview Ingestion

## Accepted Inputs

- markdown transcripts
- plain text interview dumps
- normalized imports from legacy spreadsheets

## Output Goals

- repeated pains
- repeated objections
- desired outcomes
- correction proposals for ICP, LPR, and sales script assumptions

## Rules

- transcript ingestion does not overwrite normalized contracts automatically
- corrections remain proposals until reviewed
