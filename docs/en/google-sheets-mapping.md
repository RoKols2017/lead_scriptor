# Google Sheets Mapping

Google Sheets is used only as a compatibility target.

## Export Profiles

- `marketing-homework` -> lecture 1 artifacts
- `custdev-simulated` -> lecture 2 simulated interview artifacts
- `interview-synthesis` -> findings from transcript clustering
- `lpr-discovery` -> company and decision-maker profiles
- `sales-script` -> raw stage-by-stage script
- `sales-script-styled` -> final script text

Every profile exports values only.
