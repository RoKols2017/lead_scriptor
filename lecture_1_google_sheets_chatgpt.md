## 1. Расширение ChatGPT: установка, функции и настройка

Расширение GPT for Sheets™ and Docs™ позволяет интегрировать возможности ChatGPT непосредственно в Google Таблицы, что открывает множество новых возможностей для автоматизации и оптимизации различных задач.

Для начала работы над уроком зайди в свой аккаунт Google и создай новую Google Таблицу.

### Установка расширения

Для начала работы с расширением необходимо установить его через меню дополнений в Google Sheets. Для этого:

1. В верхнем меню выбираем **Расширения — Дополнения — Установить дополнения**.
2. В поисковой строке открывшегося окошка ищем **chatgpt**.
3. Выбираем **“GPT for Sheets™ and Docs™”**. Но можно выбрать любой другой вариант.

> 💡 Интерфейс этого расширения обновился, поэтому самая актуальная информация представлена в транскрибации к уроку.

4. Нажимаем на кнопку **Установить**.

Расширение установлено.

После установки расширения открой его через меню дополнений. Выбери **Open**, а затем в появившемся окне нажми **Continue**.

### Функции и настройка расширения

Справа появится панель управления расширением. Рассмотрим основные функции:

- **Выбор модели** — доступны несколько моделей ChatGPT, включая GPT-3.5 Turbo и GPT-4. Каждая модель имеет свою стоимость.
- **Ввод API-ключа** — можно использовать расширение без указания API-ключа либо указать свой API-ключ для доступа к более экономичному использованию моделей.
- **Настройки** — здесь можно:
  - выбрать модель,
  - установить режим безопасности (**Safe Mode**), который предотвращает частую перезагрузку формул,
  - установить **Auto-replace formulas**,
  - настроить уровень креативности ответа (**temperature**) — рекомендуется установить эту настройку на **0**,
  - максимальное количество слов в ответе (**max_tokens**) — рекомендуется установить эту настройку на максимум.
- **Расширенные настройки** — можно оставить по умолчанию.
- **Кнопка List of GPT functions**, по которой открывается список доступных формул.
- **Video tutorials** — видеоуроки от разработчика расширения.
- **Пополнение баланса**.
- **API keys** — сюда можно ввести API-ключ.

В этом расширении можно обращаться к:

- OpenAI API key
- Gemini API key
- Anthropic API key
- Mistral API key

Когда мы не используем ключ OpenAI, то стоимость обработки наших сообщений будет дороже, чем если бы мы обращались к API.

Наш текущий баланс.

> ⚙ Также в расширении можно использовать свои дообученные модели.

## 2. Стоимость расширения

На сегодняшний день расширение платное.

> 📌 Поэтому важно понимать, за что мы платим деньги, как они расходуются и на что они списываются, чтобы наиболее эффективно использовать оплаченные средства.

Стоимость с использованием API и без него отличаются.

### Рассмотрим тарифы

> 📌 Используя API-ключ, мы оплачиваем как сам ключ API, так и его использование в расширении.

Подсчёт стоимости токенов можно произвести на сайте:  
`gptforwork.com/help/billing/cost-estimator`

Для подсчёта:

1. Вставляем промпт в поле **Prompt**.
2. Сверху выбираем, что именно мы используем, какую модель, с API или без него.
3. В правом верхнем углу изучаем стоимость.

У OpenAI есть свой счетчик токенов:  
`platform.openai.com/tokenizer`

Со стоимостью токенов можно ознакомиться на сайте:  
`openai.com/api/pricing`

Обработка запроса и обработка ответа оплачиваются отдельно.

> 💡 Эти счетчики для расчёта стоимости токенов могут пригодиться тебе и в других проектах, поэтому обязательно сохрани их.

Чтобы получить API-ключ:

1. Переходим по ссылке `https://platform.openai.com/api-keys`
2. Создаём ключ.
3. Вставляем в расширении.

### Автоматические перерасчеты

Если не соблюдать основные правила работы с таблицей, то расходы будут происходить постоянно.

В каких случаях Google Таблицы автоматически пересчитывают все формулы:

- При перемещении столбцов или строк, а также при сортировке строк.
- Автоматически каждые несколько часов, даже если в электронную таблицу не вносятся изменения и она закрыта.

Чтобы уберечь себя от автосписаний:

- используем по умолчанию модель **gpt-4o-mini**;
- держим включённым переключатель внизу панели настроек расширения;
- заменяем формулы их результатами (тогда формулы не будут пересчитываться).

Статья о том, как избежать перерасчетов:  
`https://gptforwork.com/help/gpt-for-sheets/gpt-functions/control-gpt-formulas/avoid-recalculations`

Инструкция:

2. В поисковой строке открывшегося окна ищем **chatgpt**.
3. Выбираем расширение **“GPT for Slides™ Docs™ Sheets™”**.
4. Нажимаем кнопку **Установить** и проходим простую регистрацию.
5. Открываем расширение: **“GPT for Slides™ Docs™ Sheets™” → Start**.
6. Заходим в настройки, нажав на шестеренку в левом верхнем углу.

## 3. Рассмотрение функционала расширения на примере

Рассмотрим на простом примере, как работает данное расширение в таблице.

Шаблон таблицы эксперта:  
`https://docs.google.com/spreadsheets/d/1OPMgKfHWXivspT1rnMUxCXvjiXV3HQZx0o9PWOAzpyA/edit?usp=sharing`

1. Создаем копию таблицы.
2. Открываем расширение.
3. Вводим промпт в ячейку C3, нажимаем Enter и ждем завершения генерации.

Формула:

```gs
=GPT(C2;$A$2)
```

Разбор:

- `=GPT` — обращение к ChatGPT.
- `C2` — условие, с которым должен работать ChatGPT.
- `A2` — описание продукта.
- `$` — фиксирует адрес ячейки, чтобы он оставался неизменным при копировании.

### Если нужно изменить модель

1. В панели справа меняем модель.
2. В поле формулы деактивируем формулу — ставим любой символ перед `=`.
3. Запускаем.
4. Удаляем символ и снова запускаем.

Если запрос остался прежним, его повторная отправка потратит меньше.

Растяни ячейку вправо, чтобы сгенерировать следующий запрос.

> ⚙ На данные задачи в расширении эксперт потратил чуть больше доллара.

В Google Таблицах важно соблюдать правила пунктуации формул. Если в формуле есть что-то лишнее, то она не будет работать.

Разберём формулу подробнее:

`GPT(prompt; [value]; [temperature]; [model])`

- `prompt` (промпт) — обязательно берётся в кавычки;
- `value` — ценность промпта;
- `temperature` (температура) — без кавычек, от 0 до 1, разделяя точкой (не `0,8`, а `0.8`). Такая температура будет использоваться только для конкретной ячейки. Температура по умолчанию выбирается на панели справа;
- `model` (модель).

Формула может содержать только две переменных.

Если мы добавляем температуру и модель в формулу, они будут действовать только для конкретной ячейки.

## 4. Основные формулы

> ⚙️ Шаблон документа для работы с формулами (формулы в шаблоне не указаны):  
> `https://docs.google.com/spreadsheets/d/1zkr5vpZjURS3j8gUM5-59ahKIfcZX08s7wxJArkRYAg/edit?usp=sharing`

Разберёмся с формулами, которые могут понадобиться чаще всего:

- использование формулы в её текущем виде — `=GPT("промпт")`;
- ссылка на ячейку — `=GPT(ячейка)`;

Промпты запоминаются только в рамках одной таблицы. Если разные пользователи используют одинаковый промпт, результаты будут разные.

- промпт, состоящий из текста и содержимого ячейки — `=GPT("промпт";ячейка)`;
- промпт, состоящий из содержимого двух ячеек — `=GPT(ячейка;ячейка)`;

Если в тексте промпта были допущены ошибки, то ChatGPT всё равно сможет понять суть запроса, если ошибка не полностью меняет смысл слова.

- составной промпт (выделение текста из нескольких ячеек) — `=GPT(ячейка:ячейка)` — тогда промпт содержится в нескольких ячейках, в выбранном диапазоне;
- сложно-составной промпт — `=GPT($A$1:$C$3;ячейка)`;
- ответ списком — `=GPT_LIST(ячейка)`. В таком случае каждый элемент ответа будет в отдельной ячейке;
- перевод — `=GPT_TRANSLATE(ячейка;"Russian")`. Язык указываем обязательно на латинице;
- очистка данных — `=GPT_EXTRACT(ячейка;ячейка)` или `=GPT_EXTRACT(ячейка;"email")`. Используется для неструктурированных данных. Будет выдано то, на что указано во второй части.
